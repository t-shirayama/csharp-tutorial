## 確認結果

- `Microsoft.DotNet.SDK.10` は Windows の winget パッケージとして公式ドキュメントに掲載されている。
- `dotnet restore` は NuGet 依存関係を復元し、`dotnet build`、`dotnet run`、`dotnet test` などで必要に応じて暗黙実行される。
- .NET 10 SDK の `dotnet new sln` は既定で `.slnx` を作るため、`.sln` 固定の説明を solution file の説明へ修正した。
- `global.json` は .NET CLI が使用する SDK バージョンを選ぶためのファイルであり、runtime target とは独立している。
- C# Dev Kit は VS Code 向けの C# 開発体験を提供する Microsoft の拡張として確認した。
