## 到達目標

- C# のコードとデータベースの境界を説明できる。
- EF Core で基本的な CRUD を実装できる。
- トランザクションが必要な場面を判断できる。
- クエリ性能、同時更新、接続文字列、repository 抽象化を実務観点で判断できる。
- Change Tracker、loading strategy、SQL 診断、DB アクセス方式の選択を説明できる。
- 生成 SQL、実行計画、index、projection、tracking を使って性能問題を切り分けられる。
- Raw SQL、Dapper 併用、DB 製品差分、複数 DbContext の判断基準を説明できる。
