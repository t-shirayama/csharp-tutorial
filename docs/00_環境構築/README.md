## 到達目標

- `dotnet --version` で SDK を確認できる。
- VS Code で C# プロジェクトを開ける。
- `dotnet new`, `dotnet build`, `dotnet run`, `dotnet test` を実行できる。
- 既存リポジトリを clone して restore / build できる。
