## 到達目標

- `dotnet --version` で SDK のバージョンを確認できる。
- VS Code で C# プロジェクトを開き、ビルドと実行ができる。
- PowerShell で `dotnet new`, `dotnet run`, `dotnet build` を使える。
- solution、project、SDK バージョン、PATH の問題を切り分けられる。
- 何をどこから入手し、インストール時にどの設定を選ぶか説明できる。
- Git repository を取得し、restore、build、test まで確認できる。
- VS Code の debug と環境情報の記録を使って、チームへ再現可能な形で相談できる。
