## 到達目標

- 標準ライブラリで解ける問題を外部パッケージに頼りすぎず解決できる。
- 日付、ファイル、JSON、HTTP の基本処理を書ける。
- 時刻、HTTP、JSON の実務上の差し替えや互換性を考慮できる。
- IDisposable と Stream を使い、resource 解放と大きなデータの扱いを説明できる。
- テキスト encoding と binary I/O の基本的な注意点を説明できる。
- `StringBuilder` と `TimeSpan` の使いどころを説明できる。
