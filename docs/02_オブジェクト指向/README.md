- 値と責務をクラスにまとめられる。
- interface を使う理由を説明できる。
- 継承より合成を選ぶ場面を判断できる。

- struct、enum、delegate、event の使いどころを説明できる。
- field、indexer、operator overload、partial、nested type の使いどころを説明できる。
